# Carrera de tortugas

Este es un juego de carreras de tortugas desarrollado en Python y utilizando la librería Turtle. El objetivo es que las tortugas avancen por la pantalla y lleguen a la línea de meta, compitiendo entre ellas.

## Requerimientos

- Python 3.x
- Librería Turtle (ya viene instalada con Python)

## Cómo jugar

1. Clona este repositorio en tu ordenador: `git clone https://github.com/LunaPy17/Carrera-De-Tortuguitas.git`
2. Ejecuta el archivo `carrera_de_tortugas.py`: `python carrera_de_tortugas.py`
3. Usa la tecla "espacio" para iniciar la carrera y la tecla "r" para reiniciar.

¡Disfruta del juego!
